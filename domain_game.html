Learning Roadmap

Step 1: Learn C# or C++
Start with a programming language. C# for Unity is beginner-friendly. C++ for Unreal is industry standard.
Step 2: Pick an Engine
Download Unity or Unreal Engine. Follow official tutorials to make a simple 2D game.
Step 3: Game Mechanics & Physics
Learn how objects interact, collision detection, and basic physics.
Step 4: 3D Basics & Assets
Learn to import 3D models (from Blender) and use textures/materials.
Step 5: Publish & Portfolio
Finish small games. Publish on itch.io. Your portfolio is your degree.
