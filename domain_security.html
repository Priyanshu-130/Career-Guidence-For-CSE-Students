Cyber Security

About the Domain

Protecting systems, networks, and programs from digital attacks. Involves Ethical Hacking, Network Security, Forensics, and Compliance.

Prerequisites

  • Networking: TCP/IP, OSI Model, DNS, VPN.
  • OS: Linux (Kali/Ubuntu) Command Line.
  • Scripting: Python/Bash for automation.
  • Tools: Wireshark, Metasploit, Nmap, Burp Suite.
  • Concepts: Cryptography, Firewalls, OWASP.

Top Companies

  • Palo Alto Networks
  • Cisco Systems
  • CrowdStrike / FireEye
  • Deloitte, EY, PwC (Cyber Auditing)
  • DRDO, ISRO (Govt Research)
  • Banks (HDFC, SBI Security Teams)

Real-Life Applications

  • Defense: Protecting national infrastructure.
  • Banking: Securing transactions and ATMs.
  • Privacy: Encrypting WhatsApp messages.
  • Anti-Virus: QuickHeal, McAfee software.
  • App Security: Bug bounty programs.

Learning Roadmap

Step 1: Networking Basics
Start with CompTIA Network+ curriculum. Learn how the internet works.
Step 2: Linux & Scripting
Get comfortable with the terminal. Learn Python for writing scripts.
Step 3: Security Fundamentals
Study CompTIA Security+. Learn about malware, phishing, encryption.
Step 4: Ethical Hacking
Learn pentesting. Use Kali Linux tools. Try Capture The Flag (CTF) challenges.
Step 5: Certification
Aim for CEH (Certified Ethical Hacker) or OSCP (Advanced).
