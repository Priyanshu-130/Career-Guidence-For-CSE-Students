Web & App Development

About the Domain

Building websites and mobile applications. Spans Frontend (what users see), Backend (server logic), and Full Stack (both).

Prerequisites

  • Frontend: HTML5, CSS3, JavaScript (ES6+).
  • Frameworks: React, Angular, or Vue.
  • Backend: Node.js, Express, or Django.
  • DB: MongoDB, PostgreSQL, Firebase.
  • Mobile: React Native, Flutter (Optional).

  • Social Media: Instagram, LinkedIn apps.
  • Streaming: YouTube, Netflix web players.
  • Productivity: Google Docs, Notion.
  • Services: Uber, Ola booking interfaces.

Learning Roadmap

Step 1: The Basics
Learn HTML for structure, CSS for styling, and JavaScript for interactivity.
Step 2: Frontend Framework
Choose React (Most popular). Learn Components, State, Hooks.
Step 3: Backend Basics
Learn Node.js + Express. Understand REST APIs and JSON.
Step 4: Database
Learn MongoDB (NoSQL) for MERN stack or PostgreSQL (SQL).
Step 5: Deployment & Git
Host on Vercel/Netlify. Use Git for version control. Build a portfolio.
