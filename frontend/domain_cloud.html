Cloud Computing

1. Domain Overview

Focuses on delivering computing services—including scalable servers, resilient storage, databases, networking, software, and analytics—over the internet (the cloud) to offer faster innovation and flexible resources.

2. Prerequisites

  • Operating Systems fundamentals
  • Linux command line basics
  • Basic computer networking
  • Scripting (Bash, Python)
  • Understanding of web architecture

3. Core Skills / Concepts

Virtualization Containerization Serverless Architecture Microservices Infrastructure as Code (IaC) CI/CD Pipelines Load Balancing High Availability

4. Tools & Technologies

Amazon Web Services (AWS) Microsoft Azure Google Cloud (GCP) Docker Kubernetes Terraform Linux Jenkins / GitHub Actions

6. Mini Projects

  • Host a highly available static website using AWS S3 and CloudFront
  • Build a Serverless API using API Gateway and AWS Lambda
  • Deploy a containerized web application using Docker and Kubernetes
  • Automate infrastructure deployment using Terraform

7. Job Roles

Cloud Engineer DevOps Engineer Cloud Solutions Architect Site Reliability Engineer (SRE) Cloud Security Engineer
