Data Science

1. Domain Overview

Focuses on extracting insights from data using statistics, visualization, analytics, and predictive techniques.

2. Prerequisites

  • Python basics
  • Statistics
  • Probability
  • SQL basics
  • Spreadsheet basics

3. Core Skills / Concepts

Data cleaning Exploratory data analysis Data visualization Statistical analysis SQL querying Reporting and dashboarding Predictive analytics basics

4. Tools & Technologies

Python Pandas NumPy Matplotlib Seaborn SQL Excel Power BI Tableau

Step 2

Learn predictive analytics basics

Step 6
